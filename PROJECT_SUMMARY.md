1. **Build vs. Integrate Decision**
   - Odoo is powerful but too generic
   - White-labeling still feels like Odoo
   - Building our own gives 100% control
   - Our 67 bots are the differentiator
   - **Decision: BUILD OUR OWN!**

2. **Market Opportunity**
   - South African SME market underserved
   - Existing ERPs expensive and complex
   - AI automation is the killer feature
   - $100M+ market opportunity

3. **Technology Stack**
   - FastAPI = Modern, fast, well-documented
   - React = Industry standard, huge ecosystem
   - PostgreSQL = Enterprise-grade reliability
   - Docker = Easy deployment

### Technical Lessons

1. **Multi-Tenancy Pattern**
   - Simple company_id approach works
   - Row-level security
   - Easy to understand and maintain

2. **AI-First Design**
   - Added bot_* fields from day 1
   - Designed for automation
   - Human review layer built-in

3. **South African Focus**
   - BBBEE compliance built-in
   - SARS integration planned
   - Local regulations in schema
   - ZAR as default currency
