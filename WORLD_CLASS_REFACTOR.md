## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- Python 3.9+
- PostgreSQL 13+
- Redis (optional, for caching)

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 3000
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev -- --port 12008
```

### Access Points
- **Homepage**: http://localhost:12008
- **Modern Login**: http://localhost:12008/modern-login
- **Enhanced Dashboard**: http://localhost:12008/enhanced-dashboard
- **AI Bot Interface**: http://localhost:12008/ai-bot
- **API Documentation**: http://localhost:3000/docs
