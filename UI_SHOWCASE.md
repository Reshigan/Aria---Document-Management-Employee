## 🎬 Animation Showcase

### Entrance Animations
1. **Fade In** (0.5s)
   - Opacity: 0 → 1
   - Used for: General content

2. **Slide Up** (0.5s)
   - Y: +20px → 0, Opacity: 0 → 1
   - Used for: Cards, messages, sections

3. **Slide Down** (0.5s)
   - Y: -20px → 0, Opacity: 0 → 1
   - Used for: Headers, dropdowns

4. **Slide In (Left/Right)** (0.5s)
   - X: ±20px → 0, Opacity: 0 → 1
   - Used for: Side panels, navigation

5. **Scale In** (0.4s)
   - Scale: 0.9 → 1, Opacity: 0 → 1
   - Used for: Modals, popups

### Continuous Animations
1. **Float** (3s infinite)
   - Y: 0 → -20px → 0
   - Used for: ARIA avatar, decorative elements

2. **Pulse** (2s infinite)
   - Opacity: 1 → 0.6 → 1
   - Used for: Status indicators, loading states

3. **Glow** (2s infinite)
   - Shadow: 20px blur → 40px blur → 20px blur
   - Used for: Active elements, highlights

4. **Shimmer** (1.5s infinite)
   - Background position sweep
   - Used for: Loading placeholders

5. **Bounce** (1.4s infinite)
   - Y: 0 → -10px → 0
   - Used for: Loading dots, attention grabbers

## 🎨 Design Patterns

### Glass Morphism
```
Background: rgba(255, 255, 255, 0.7)
Backdrop Filter: blur(10px)
Border: 1px solid rgba(255, 255, 255, 0.3)
```
Used for: Overlay cards, feature highlights

### Gradient Backgrounds
```
Hero: Blue 50 → Purple 50 → Cyan 50
Cards: Light gradient for depth
Buttons: Blue → Purple (primary action)
```

### Card Hierarchy
1. **Subtle**: White with light shadow
2. **Elevated**: White with medium shadow + hover
3. **Featured**: Gradient background + strong shadow

### Button States
```
Default:  Solid or outlined
Hover:    Lift + shadow increase
Active:   Slight scale down
Disabled: 50% opacity
Loading:  Spinner + disabled
```
